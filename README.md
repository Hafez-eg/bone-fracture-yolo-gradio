# Bone Fracture Detection using YOLOv8

This project detects bone fractures from X-ray images using YOLOv8 and provides a Gradio web interface.

## Project Structure
- `train.ipynb` : Colab notebook for dataset download, visualization, training, evaluation, and Gradio deployment.
- `app.py` : Standalone Gradio app for inference.
- `data.yaml` : Dataset configuration for YOLOv8.
- `requirements.txt` : Required Python packages.
